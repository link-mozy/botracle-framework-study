# Technical Feature Analysis (기술적 특징 분석)

## 프로젝트 개요

BOTracle 프래임워크 기반의 기술적 특징 분석 (SGAN) 프로젝트 입니다.


## 개념

**SGAN (Semi-Supervised Generative Adversarial Network)**을 사용하여 기술적 특징을 분석합니다. 레이블이 있는 데이터와 없는 데이터를 모두 활용할 수 있습니다.

## SGAN 활용 방법

### 1단계 : 특징 추출 (Feature Extraction)

사용자의 행동 데이터를 수자로 변환합니다.

- 마우스 움직임 속도
- 클릭 간격
- 키보드 타이핑 리듬
- 페이지 머무는 시간

### 2단계 : 비교 학습 (Siamese 부분)

두 개의 행동 패턴을 비교합니다.

```plaintext
사용자 A의 패턴 ──┐
                  ├─→ 비슷한가? (유사도 점수 계산)
사용자 B의 패턴 ──┘
```

**예시**
- 정상 사용자끼리 비교 → 유사도 낮음 (각자 다른 패턴)
- 봇 끼리 비교 → 유사도 높음 (똑같은 스크립트 사용)

### 3단계 : 학습 (GAN 부분)

- 생성자(Generator) : 봇 처럼 보이는 가짜 패턴을 생성
- 판별자(Discriminator) : 진짜 봇인지 가짜 패턴인지 구별

이 둘이 계속 경쟁하면서 판별 능력을 향상 시킵니다.
